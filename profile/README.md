## 🚀 Key Projects

Our ecosystem consists of specialized components grouped into tactical categories.

### 📬 Mail Infrastructure
*   **[madmail](https://github.com/themadorg/madmail)**: A specialized maddy fork delivering high-performance email for Delta Chat.
*   **[madexchanger](https://github.com/themadorg/madexchanger)**: Secure HTTPS mail relay for efficient message routing between madmail servers.

### 🛠 Management Tools
*   **[madmail-admin-web](https://github.com/themadorg/madmail-admin-web)**: A modern, intuitive web dashboard for managing madmail configurations.

### 📚 Knowledge & Community
*   **[deltachat-wiki](https://github.com/themadorg/deltachat-wiki)**: Comprehensive documentation and wiki for the Delta Chat ecosystem experiments.
